API Exploration

API Description/What It Does:

This API is a current weather API. It provides information about the current weather an area of interest by ZIP code. In this case, the area of interest is Boston.

Website Where It Was Found:

openweathermap.org

Options Used For API Request:

The API request option used was the one that called weather information about the city of Boston by ZIP code.

Two Applications Where This API Would Be Helpful:

1) Helpful for people in Boston (or another area) to see the current weather to decide how they should dress for the weather.

2) Helpful for people in Boston (or another area) to see whether the current situation is good for traveling outside.
